## Presentación

En este proyecto trabajamos 4 datasets que tratan sobre el accidente del crucero **Titanic** con el fin de predecir la probabilidad de que una persona haya sobrevido al accidente o no.

## Tecnologías

Python:
  * Pandas
  * Numpy
  * Seaborn
  * Matplotlib

Predicciones:
  * Scikit-learn
